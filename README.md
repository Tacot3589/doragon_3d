### Doragon - My first working sumo robot
Sumo robot build by me! Built for roborave 2024:
- 20x18cm footprint
- about 750g
- 2 Distance sensors 
- 2 line sensors
- motors run on 24v
- about 2.5:1 gear ratio 

Pros:
 - it is hella fast
 - idduno men it was my first somewhat working sumo robot
 - it drifted really well :)

Cons:
- to fast motors, with not enough torque
- to little mass on the wheels, so little to no grip

Accomplishments:
 - top 8 place in robotic competition (20x20cm 3kg) in Rzeszów 2025
